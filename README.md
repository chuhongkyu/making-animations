# 3D모델 애니메이션 만들기

- 프로젝트에 node_modules를 설치합니다.
```
npm i 
```

- npm start로 프로젝트를 실행합니다. 
    - 서버와 같이 실행됨.
    - ("start": "concurrently \"npm run server\" \"npm run dev\"")
    - 클라이언트: locallhost:9000
    - 서버: locallhost:3000
```
npm start
```
